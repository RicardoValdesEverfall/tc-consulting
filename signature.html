To install (easiest method)

  1. Select the entire preview block above (click just before the logo, drag to just after the email address). Press Ctrl+C (Windows) or Cmd+C (Mac).
  2. Open your email client's signature settings (see below).
  3. Paste with Ctrl+V / Cmd+V. Save.

IONOS Webmail

  1. Log into IONOS webmail.
  2. Settings (gear icon) → Mail → Signatures.
  3. Click Add new signature. Name it "T.C Consulting".
  4. Make sure the editor toolbar shows the HTML/formatted mode (not plain text).
  5. Paste the preview block. Save.
  6. Assign the signature to each T.C mailbox (info@, ricardo@, bookings@).

Gmail (if you forward TC mail through Gmail)

  1. Settings (gear) → See all settings → General tab.
  2. Scroll to Signature → Create new.
  3. Paste the preview block into the signature box.
  4. Set "Signature defaults" to use it for new mail and replies.
  5. Click Save Changes at the bottom.

Outlook (desktop)

  1. File → Options → Mail → Signatures…
  2. Click New. Name it "T.C Consulting".
  3. Paste the preview block.
  4. Set as default for new messages and replies.

Apple Mail (macOS)

  1. Mail → Settings → Signatures.
  2. Pick your account, click + to add a new signature, name it.
  3. Uncheck "Always match my default message font".
  4. Paste the preview block.

If the logo doesn't display in some clients

A few corporate spam filters and older Outlook versions strip remote images by default. Recipients can click "Show images" to reveal it. The logo is served from https://tccgroup.ca/assets/logo.png (your own domain) — that's the most trusted source possible.
